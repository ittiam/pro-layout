现代浏览器及 IE10

## 使用

### clone

```bash
$ git clone https://gitlab.cloud4water.com/WPG_SRC_WEB/pro-layout
```

### yarn

```bash
$ yarn install
$ yarn dev
```

### or npm

```
$ npm install
$ npm run dev
```

更多信息参考 [使用文档](https://gitlab.cloud4water.com/WPG_SRC_WEB/pro-layout/docs)
